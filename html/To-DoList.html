Description

This is small To-Do list application that I developped using Nodejs, Nextjs, MUI, and MongoDB. Users can add, edit, and remove "to-do" items from their browser to updatea database which is hosted locally. This web app currently does not include any security checks or input handling as it was intended to be run locally by a single user.
